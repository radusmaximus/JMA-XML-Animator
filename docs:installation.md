# Installation

## Install the Blender add-on

1. Open Blender.
2. Go to **Edit > Preferences > Add-ons**.
3. Click **Install...**.
4. Select `src/jma_xml_animator.py`.
5. Enable the add-on.
6. Open the 3D Viewport and press **N** to open the sidebar.
7. Open the **JMA Animator** tab.

## NumPy

The add-on uses NumPy for numerical operations. In most manuscript workflow environments, NumPy was available in the Blender Python environment.

If NumPy is missing, install it into Blender's Python environment according to the Blender version and operating system used.

## Notes

This repository does not include raw clinical XML, STL, OBJ, or PLY files. Use only anonymised or ethically approved datasets in accordance with local institutional requirements.
