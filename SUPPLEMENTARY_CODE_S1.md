# Supplementary Code S1

## JMA XML Animator v1.0.0

This supplementary file provides repository information for **JMA XML Animator v1.0.0**, the Python add-on for Blender used in the manuscript for XML import, SVD-based mandibular motion reconstruction, threshold-based frame selection, animation, and occlusal contact-area computation.

## Public Repository

The custom Python add-on is available through a public GitHub repository:

https://github.com/radusmaximus/JMA-XML-Animator

The repository includes:

- the JMA XML Animator v1.0.0 Python add-on
- documentation
- software version requirements
- installation instructions
- usage instructions
- license information

## Data Privacy

The repository does not include raw patient datasets, STL/OBJ/PLY models, XML kinematic recordings, screenshots containing patient data, or example workflow files derived from clinical cases. These data are not publicly available due to privacy and ethical restrictions.

## Intended Use

The add-on is provided for research and methodological validation purposes. It is not intended as a standalone clinical decision-support system and should not be used as the sole basis for diagnosis, treatment planning, prosthodontic design, or occlusal adjustment.
