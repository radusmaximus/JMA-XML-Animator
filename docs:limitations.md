# Limitations

JMA XML Animator is a research and methodological validation tool developed for analysis of exported mandibular motion data.

The add-on does not independently validate the absolute hardware accuracy of Zebris JMA, Medit i700, or any other acquisition system.

SVD residuals generated within the workflow should be interpreted as internal rigid-body fitting residuals, not as direct measures of clinical accuracy.

The repository does not include raw clinical XML, STL, OBJ, or PLY files because these datasets contain patient-related anatomical and kinematic information and are subject to privacy and ethical restrictions.

The software should not be used as a standalone clinical decision-support system and should not be used as the sole basis for diagnosis, treatment planning, prosthodontic design, or occlusal adjustment.
