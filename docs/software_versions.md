# Software Versions

The manuscript workflow used the following software environment:

- Add-on: JMA XML Animator v1.0.0
- Blender: 4.5 LTS
- Blender4Dental: 1.1.99
- Python: Blender-bundled Python environment
- NumPy: required for numerical operations

Additional software used for workflow verification in the study:

- Medit Occlusion Analyzer 1.0.2
- Tizian Function Pro 3.0.2
- Planmeca CAD Premium 3.0

The add-on metadata specifies Blender 3.0.0 as the minimum Blender version; however, the manuscript workflow was developed and tested in Blender 4.5 LTS.
